<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites


* Run Test;
npm run test

Seed DB;
npm run seed

Start Server;
npm start server

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:caflores31/psychic-engine.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
